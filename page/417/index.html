    How to add a map to any blog page

    I came across this interesting bit in LifeHacker: WikiMapia - The geographical wikipedia - Lifehacker. It teaches that using WikiMapia, you can add a map to any web page in 30 seconds. If that claim is true, then you should be seeing a map of where I was born below. If you don't see that map, then ignore this post, please 🙂 http://wikimapia.org/s/#y=12212522&x=-68950195&z=10&l=0&m=a p.s. It worked!!! Just shows that there's nothing you can't get from the internet, and that Google Maps is unbelievably cool! Technorati Tags: google, wikimapia
