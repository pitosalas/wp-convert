(**Web site except:** SSH (secure shell) is good for remotely managing machines using a secure connection. Typically you will log in to a server using the command-line `ssh` tool, or something like PuTTy or MobaXTerm. This guide will show you how to use Python to connect and run commands over SSH using the Paramiko package. - [Paramiko Documentation](https://docs.paramiko.org/en/stable/) - [Paramiko Source on GitHub](https://github.com/paramiko/paramiko))
